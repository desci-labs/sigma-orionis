These CASA/Python scripts were used to clean the ALMA data, extract FITS images, and measure fluxes. There is a continuum script for each source and a line script for the six gas detections. The line scripts for the gas detections include sections for creating the moment maps in Figure 3 of Ansdell et al. 2017. For the many other gas non-detections, there are scripts (e.g., extract_line_vis.py) that automate their bulk processing. The CASA version 4.5.3 that was used for the processing is very out of date; the scripts as-is will not work with the current version of CASA.
